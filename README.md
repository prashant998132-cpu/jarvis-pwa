# Jarvis PWA

A free, offline-first AI companion for Rewa MP. Built with love by Prashant Patel.

## Features
- Voice Assistant (Hindi + English)
- Offline AI (WebLLM + Transformers.js)
- Long-term Memory (IndexedDB)
- Proactive Alerts (Battery, Time-based)
- Glassmorphism UI (Tailwind + Framer Motion)

## Setup
```bash
npm install
npm run dev
```
